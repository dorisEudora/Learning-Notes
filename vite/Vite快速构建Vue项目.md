# Vite快速构建Vue项目

## 1.语法

1. npm(cnpm) init @vitejs/app <project name> 

2. 创建项目vue3-demo：npm init @vitejs/app  [project name]

   or yarn create @vitejs/app [project name]

3. 运行项目

   1. cd vue3-demo

   2. npm install

   3. npm run dev
